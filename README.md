# 🚀 Foresight - AI Powered Demand & Inventory Intelligence Platform

## 📌 Project Overview
An AI-powered Data Analytics and Machine Learning project designed to improve inventory management through demand forecasting and business intelligence.


## 🎯 Objectives
- 📊 Analyze retail sales data
- 🧹 Clean and preprocess raw data
- 📈 Perform Exploratory Data Analysis (EDA)
- 🤖 Forecast future product demand
- 📦 Predict stockout and overstock situations
- 📉 Build interactive Power BI dashboards
- 💡 Generate actionable business insights


## ✨ Key Features
- ✅ Data Cleaning & Preprocessing
- ✅ Exploratory Data Analysis (EDA)
- ✅ Customer Behavior Analysis
- ✅ Product Performance Analysis
- ✅ Sales Trend Analysis
- ✅ Demand Forecasting using Machine Learning
- ✅ Inventory Optimization
- ✅ Interactive Power BI Dashboard
